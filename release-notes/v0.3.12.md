# cchahatui v0.3.12

v0.3.12 maps to the non-destructive forgotten project records slice.

## Changes

- Treat project forgetting as a hidden-project operation instead of only removing saved project index entries.
- Hide session-derived project groups after removal without deleting JSONL transcripts or real workspace folders.
- Keep hidden project state inside the cchahatui-owned config area.
- Respect hidden projects in normal and project-filtered session lists.
- Adding the same project again restores it to the project/session list.
