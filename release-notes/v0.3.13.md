# cchahatui v0.3.13

v0.3.13 maps to the restricted project memory hardening slice.

## Changes

- Block project memory persistence for temporary thoughts, failed attempts, API keys, OAuth material, private filesystem paths, and raw chat transcript lines.
- Sanitize legacy project memory during localStorage normalization so old restricted content is not revived.
- Sanitize project memory again before injecting model context to guard already-loaded records.
- Keep allowed project facts, decisions, and open tasks while skipping restricted lines.
- Show a warning toast when restricted memory content is skipped.
