# Claude Code Haha v0.3.0

这是一个项目工作台版本，重点把桌面端左侧导航从普通会话列表推进到更接近 Codex 的项目视图：用户可以按项目查看会话、直接在项目下新建对话，并为同一项目内的不同对话保存共享记忆。

本版本也补上了浏览器开发模式下的启动回退逻辑，避免前端 Vite 地址被误当成本地 API 服务，导致 `/health` 返回 HTML 后出现“本地服务启动失败”。

## Highlight

- **项目侧边栏**：左侧会话列表按项目目录聚合，项目名、项目会话和运行状态更容易扫描。
- **项目管理入口**：项目组内可以直接新建会话，并为后续项目管理扩展保留清晰入口。
- **项目记忆**：支持为项目保存共享记忆，同一项目下不同对话会自动带入项目上下文。
- **快捷键**：新增常用桌面快捷键，支持快速新建会话、聚焦输入、折叠侧边栏、打开项目记忆和切换前几个会话。
- **启动健康检查修复**：浏览器开发模式会在 Vite origin 返回非 JSON `/health` 时回退到默认本地服务地址，避免把前端页面误判为 API 服务。

## 问题修复

- 修复 `http://127.0.0.1:<vite-port>/health` 返回 HTML 时反复重试并最终报错的问题。
- 本地服务默认回退到 `http://127.0.0.1:3456`，显式传入的本地服务地址仍保持严格校验，不会静默切换。
- 项目记忆持久化增加迁移入口，避免旧本地 UI 状态升级时丢失或污染用户配置。
