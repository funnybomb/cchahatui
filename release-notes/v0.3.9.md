# cchahatui v0.3.9

v0.3.9 maps to issue #8: Project Workspace 1.0 project identity foundation.

## Changes

- Added stable project identity metadata based on canonical absolute paths.
- Added Git-aware project identity fields for repo root, redacted origin URL, repo name, and branch.
- Kept legacy `projectPath` compatibility for sessions, recent projects, sidebar state, and project memory.
- Preserved cchahatui project index isolation from shared `~/.claude` project content.
