# cchahatui v0.3.17

v0.3.17 maps to the desktop updater concurrency hardening slice.

## Changes

- Deduplicate concurrent desktop update checks through one active native updater request.
- Reuse the in-flight check result for simultaneous manual and silent refresh checks.
- Avoid starting new updater checks while an update is downloading or the app is restarting.
- Clear the in-flight guard after failures so later checks can recover normally.
