# cchahatui v0.3.16

v0.3.16 maps to the desktop updater auto-refresh slice.

## Changes

- Add periodic silent desktop update checks while the Tauri app stays open.
- Keep browser/H5 runtime free of native updater initialization and polling.
- Skip refresh checks while the updater is already checking, downloading, or restarting.
- Clean up the polling timer when the updater component unmounts.

- `bun run check:desktop`
