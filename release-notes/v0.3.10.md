# cchahatui v0.3.10

v0.3.10 maps to the settings and project folder picker refinement slice.

## Changes

- Kept DeepSeek as the default reasoning-effort guidance in General settings.
- Added provider-aware reasoning-effort copy for OpenAI, Claude, and other configured providers.
- Switched the project folder selection flow to prefer the native system folder picker.
- Added a localhost-only server folder chooser so the browser development app can open Finder on macOS.
- Kept the old in-app directory browser as a fallback when native folder selection is unavailable.
