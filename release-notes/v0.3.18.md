# cchahatui v0.3.18

v0.3.18 maps to the project picker privacy hardening slice.

## Changes

- Redact macOS, Linux, and Windows user home-directory prefixes in project picker display paths.
- Show `~/...` or `~\...` in recent-project subtitles and selected project hover text instead of private usernames.
- Keep shared/non-home paths readable.
- Preserve raw project paths for selection, project registration, filtering, and removal behavior.
