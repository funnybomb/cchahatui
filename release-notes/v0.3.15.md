# cchahatui v0.3.15

v0.3.15 maps to the private project label hardening slice.

## Changes

- Hide private path-derived project slugs from primary sidebar, filter, status bar, memory, and MCP badge labels.
- Prefer real folder names when a session has both sanitized `projectPath` and real `workDir`.
- Deduplicate project groups when the same project appears through both a sanitized key and a real path.
- Keep URL-like session titles visible as session titles while project labels stay safe.
- Preserve missing-directory project state and counts after safer display-name normalization.
