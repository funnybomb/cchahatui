# cchahatui v0.3.11

v0.3.11 maps to the browser startup server discovery hardening slice.

## Changes

- Prefer the default local backend server over the Vite frontend origin when the browser app runs on a different localhost port.
- Keep same-origin H5 behavior intact when the browser shell is served directly by the backend server.
- Fall back to same-origin health checks if the default local backend is unavailable.
- Add concrete recovery hints to the startup failure panel, including commands for starting the backend, starting Vite, and checking backend health.
